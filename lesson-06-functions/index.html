What are Functions?

Functions are like tools in a toolbox - reusable pieces of code that perform specific tasks. Instead of writing the same code over and over, you create a function once and use it whenever needed!

Real World Example:

Making Coffee:

  • ‚ùå Without functions: Write "grind beans, add water, brew, serve" every time
  • ‚úÖ With functions: Create a "makeCoffee()" function and just call it!

Functions let you organize code like organizing tools - each has a specific purpose!

Function Declaration - Creating Your Tools

Creating a function is like building a new tool. You define what it does once, then use it many times.

Basic Function Structure:

function greetUser() {
    console.log("Hello! Welcome to our website!");
    return "Greeting complete";
}

Parameters - Giving Tools Information

Parameters are like settings on a tool - they let you customize how the function works each time you use it.

Function with Parameters:

function greetPerson(name, age) {
    console.log(`Hello ${name}! You are ${age} years old.`);
    return `Greeted ${name}`;
}

Return Values - Getting Results Back

Return values are like the output of a tool - the result you get after the function does its work.

Functions that Return Values:

function addNumbers(a, b) {
    const result = a + b;
    return result; // Send the result back
}

const sum = addNumbers(5, 3); // sum = 8

Arrow Functions - Modern Syntax

Arrow functions are a shorter, modern way to write functions. They're like compact tools that do the same job!

Arrow Function Syntax:

// Traditional function
function double(x) {
    return x * 2;
}

// Arrow function (shorter!)
const double = (x) => x * 2;

// Even shorter for single parameter
const double = x => x * 2;

Scope determines where variables can be used. It's like having different toolboxes for different rooms!

Global Scope

Variables available everywhere

Like a tool you carry with you

Function Scope

Variables only inside the function

function test() { let localVar = "Only here!"; }
