What is Programming?

Programming is like giving instructions to a computer. Just like you might give directions to a friend, you give step-by-step instructions to a computer to make it do what you want.

Real World Example:

Making Coffee Instructions:

  1. Fill kettle with water
  2. Turn on kettle
  3. Wait for water to boil
  4. Add coffee to cup
  5. Pour hot water

Programming is similar: We write step-by-step instructions for computers!

What is JavaScript?

JavaScript is a programming language that makes websites interactive. It's what makes buttons clickable, forms submittable, and pages dynamic!

Your First JavaScript Code

Let's write your very first JavaScript program. The traditional first program is called "Hello World":

JavaScript Code:

console.log("Hello, World!");

This tells the computer to display "Hello, World!" in the console (developer tools).

How JavaScript Works in Web Pages

JavaScript works together with HTML and CSS:

  • HTML - The structure (like the skeleton of a house)
  • CSS - The styling (like paint and decorations)
  • JavaScript - The behavior (like electricity that makes things work)
