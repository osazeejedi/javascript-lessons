The most basic conditional statement. It runs code only if a condition is true.

    // Code runs only if condition is true

If-Else Statements

When you want to do one thing if a condition is true, and something else if it's false.

If-Else Structure:

if (condition) {
    // Code runs if condition is true
} else {
    // Code runs if condition is false
}

Multiple Conditions (Else If)

When you have more than two possible outcomes, use else if to check multiple conditions.

Multiple Conditions:

if (condition1) {
    // First option
} else if (condition2) {
    // Second option
} else if (condition3) {
    // Third option
} else {
    // Default option
}

Logical Operators

Combine multiple conditions using logical operators to create more complex decision-making.

AND (&&)

Both conditions must be true

age >= 18 && hasLicense

Can drive if 18+ AND has license

OR (||)

At least one condition must be true

isWeekend || isHoliday

No work if weekend OR holiday

NOT (!)

Reverses the condition

!isRaining

Go outside if NOT raining

Switch Statements

When you have many possible values for a single variable, switch statements can be cleaner than multiple if-else statements.

Switch Statement Structure:

switch (variable) {
    case 'value1':
        // Code for value1
        break;
    case 'value2':
        // Code for value2
        break;
    default:
        // Default code
}

Game Logic Example:

function makeChoice(choice) {
    if (choice === 'enter') {
        if (playerHealth > 50) {
            // Brave enough to enter
            showStory("You push open the heavy door...");
        } else {
            // Too weak to enter
            showStory("You're too weak to push the door...");
        }
    } else if (choice === 'around') {
        // Explore around the castle
        let randomEvent = Math.random();
        if (randomEvent > 0.5) {
            // Good outcome
            increaseScore(10);
        } else {
            // Bad outcome
            decreaseHealth(10);
        }
    } else {
        // Leave the castle
        showStory("You decide discretion is the better part of valor...");
    }
}
