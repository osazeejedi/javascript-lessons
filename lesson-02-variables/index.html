What are Variables?

Think of variables as labeled boxes where you can store information. Just like you might have a box labeled "Photos" to store your pictures, variables have names and store data.

Creating Variables

In JavaScript, we create variables using the let keyword:

Basic Variable Creation:

let userName = "Alice";
let age = 28;
let isStudent = true;

This creates three "boxes" with different types of information!

Using Variables

Once you store information in a variable, you can use it anywhere in your program:

Using Variables:

let firstName = "Sarah";
let lastName = "Johnson";
let fullName = firstName + " " + lastName;
console.log("Hello, " + fullName + "!");

This combines variables to create new information!

Changing Variables

Variables can change their values - that's why they're called "variables"!

Changing Values:

let score = 0;        // Start with 0
score = 10;           // Change to 10
score = score + 5;    // Add 5, now it's 15
