What are Loops?

Loops are one of the most powerful features in programming. They allow you to repeat actions efficiently without writing the same code over and over again.

Real World Example:

Making 100 Paper Airplanes:

  • ‚ùå Without loops: Write "fold paper, make airplane" 100 times
  • ‚úÖ With loops: Write "repeat 100 times: fold paper, make airplane"

JavaScript loops work the same way - write once, repeat many times!

For Loops - Counting Made Easy

For loops are perfect when you know exactly how many times you want to repeat something.

For Loop Structure:

for (let i = 0; i < 10; i++) {
    // This code runs 10 times
    console.log("Count: " + i);
}

While Loops - Conditional Repetition

While loops continue running as long as a condition is true. Perfect for situations where you don't know exactly how many times to repeat.

While Loop Structure:

let count = 0;
while (count < 5) {
    console.log("Count: " + count);
    count++; // Don't forget to update!
}

Loop Control - Break and Continue

Sometimes you need to exit a loop early or skip certain iterations. That's where break and continue come in!

Break Statement

Exits the loop completely

if (condition) break;

Stop when you find what you're looking for

Continue Statement

Skips to the next iteration

if (condition) continue;

Skip certain values but keep going

Nested Loops - Loops Within Loops

You can put loops inside other loops to create patterns, tables, and complex structures.

Nested Loop Example:

for (let row = 1; row <= 3; row++) {
    for (let col = 1; col <= 3; col++) {
        console.log(`Row ${row}, Column ${col}`);
    }
}

üí° Loop Mastery Tips:

  • Always make sure your loop condition will eventually become false
  • Use for loops when you know the number of iterations
  • Use while loops when the condition depends on changing data
  • Be careful with nested loops - they can slow down your program
  • Use break and continue to make your loops more efficient
